Overview (Capstone Design)

'strawberry-harvest-ai'는 캡스톤 디자인 프로젝트로, 컴퓨터 비전과 로보틱스 기술을 결합하여 딸기 수확 보조 시스템을 구축하는 것을 목표로 했습니다.

Intel RealSense 카메라로 딸기를 탐지하고 숙도를 3단계(설익음, 익음, 과숙)로 분류한 뒤, ROS(Robot Operating System)를 기반으로 로봇팔(Indy7, OpenMANIPULATOR-X)이 자동으로 익은 딸기만 수확하도록 '탐지-판단-제어' 파이프라인을 통합 구현했습니다.

Key Features

1. Ripeness Classification (Detection)

입력된 RGB 이미지에서 딸기 영역(Segmentation)을 먼저 검출합니다. 이후 각 딸기 영역의 픽셀을 RGB에서 HSV 색 공간으로 변환하고, H(색상) 값을 분석하여 빨간색 픽셀의 비율을 계산합니다. 이 비율을 기준으로 딸기의 숙성도를 3단계(설익음, 익음, 과숙)로 분류합니다.

[Detection] 딸기 영역 검출(BBox) 결과

[Logic] RGB-HSV 변환 기반의 숙성도 분류 알고리즘

2. Robotic Arm Control (Control Stack)

분류 결과 '익음'으로 판단된 딸기들에 대해, Depth 데이터를 활용하여 각 딸기까지의 3D 거리와 수확을 위한 접근 각도를 계산합니다. 이 좌표 데이터는 NVIDIA Jetson Nano에서 ROS 토픽으로 발행(publish)됩니다.

제어 노드(Node)는 이 좌표를 구독(subscribe)하여 Indy7 (TCP/IP) 및 OpenMANIPULATOR-X (UART TTL) 로봇팔을 구동시킬 Gripper/Joint 제어 명령(CMD)을 생성하고 전송합니다.

[Hardware] 사용된 로봇팔 (Neuromeka Indy7, OpenMANIPULATOR-X)

[Action] 탐지된 딸기 좌표로 로봇팔이 이동한 실제 동작 모습

[Control] Depth 카메라를 이용한 수확 지점 거리 및 각도 추정
